在仓库所有者拥有相关权利的范围内，开源星榜生成的排行 JSON、统计窗口、排名派生字段和原创可视化采用 [Creative Commons Attribution 4.0 International](https://creativecommons.org/licenses/by/4.0/)（CC BY 4.0）供他人下载、缓存、分析和再发布。

推荐采用以下署名：

> 数据来源：开源星榜（https://728792899-create.github.io/open-source-star-rank/），基于 GH Archive 与 GitHub 公共 API；统计口径见 https://728792899-create.github.io/open-source-star-rank/methodology/。

复用公开数据时应当：

1. 提供开源星榜站点或 GitHub 仓库链接，并链接数据口径页面。
2. 标明数据日期、统计窗口和所使用的榜单类型。
3. 不把观测结果描述为 GitHub 官方统计或无遗漏的全站净增数据。
4. 修改数据、筛选范围或排名算法时，明确标注已经过二次处理。
